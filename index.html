Can we actually design in browser?

I think so. You should too.

We are gaining so much power each day with new tools in the browser, and it is becoming a nuisance to be working in so many places so inefficiently. One of those places, is layout. The browser's box model, makes designing layouts in the browser something that would be magical, if only we had the tools to manipulate the dom efficiently. Well, here's my first stab at it.

We know JavaScript, we know the browser like the back of our hands. It's time to start building our own tools and leaving tools like photoshop in their rightful place, which is not layout. Here is my start at getting there. Play around with the inputs at the top of the page to change global settings to the layout.

Hover over grid elements and click on their configure icon to edit the number of columns they consume. I have lots of ideas on how to improve this and to make it a viable tool for design, though it's already proving very useful for me. Tell me your ideas, so I can make it a better tool for you too!
